#se nao indiar destino, o repositorio é criado no diretorio atual com o nome do repositorio

# atualizar repositorio
git add ficheiros

exemplo: git add ficheiro.txt
         git add . (o ponto são todos os ficheiros)

git commit -m "mensagem" 

exemplo:
git commit -m "adicionado ficheiro.txt"
git commit -m "corrigido bug"

git push

exemplo:
git push origin master
ou apenas git push se é para enviar branch atual



Como atualizar o pacote que já fiz download??
git pull

e se quiser especificar uma branch?
git pull origin branch


O que é um branch?
Branch é uma ramificação do projeto, ou seja, é uma cópia do projeto que pode ser alterada sem afetar o projeto principal.
O branch principal é o master ou main, mas podemos criar outros branches para desenvolver novas funcionalidades sem afetar o projeto principal.

Depois de feita a branch, podemos fazer o merge da branch com o projeto principal.
Isto é util para nao mexer no main e para trabalhar ao mesmo tempo.

Comando para fazer branch:
git branch nome_da_branch

Comando para mudar de branch:
git checkout nome_da_branch

comando para fazer merge:
git merge nome_da_branch

Por fim, podemos apagar a branch:
git branch -d nome_da_branch

Por vezes podemos ter conflitos ao fazer merge, isso acontece quando duas pessoas alteram a mesma linha de código.
Para resolver conflitos, podemos abrir o ficheiro e escolher a linha que queremos manter.
Depois de resolver os conflitos, podemos fazer o commit e o push.


Exemplo geral:
alterei um ficheiro e quero atualizar o meu trabalho no repositorio

git add . (ou git add ficheiro.txt)
git commit -m "create a new file"
git push
#e voilá, o ficheiro foi atualizado no repositorio
